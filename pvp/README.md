# 🌻 植物大战 PvP (Plant vs Plant)

基于 C++ 和 EasyX 图形库实现的植物大战僵尸风格双人对战游戏。

## 功能特性

- 🎮 **双人本地对战**：1P 与 2P 在同一键盘上对战
- 🌿 **可选角色**：豌豆射手（Peashooter）和向日葵（Sunflower），各有独特技能
- 💥 **丰富的战斗系统**：
  - 普通攻击与特殊技能（EX 攻击）
  - 子弹碰撞检测与爆炸特效
  - 血条 UI 显示
- 🏃 **角色动作**：待机、奔跑、攻击、死亡等多套动画
- 🗺️ **平台跳跃**：多层平台地形，支持跳跃和重力
- 📷 **摄像机系统**：画面震动效果增强打击感
- 🎵 **完整音效**：BGM、攻击音效、UI 音效
- 🎨 **粒子效果**：奔跑烟尘、跳跃和落地特效
- 🏆 **胜负判定**：角色死亡后显示胜利画面
- 📋 **场景管理**：菜单场景 → 角色选择场景 → 游戏场景

## 技术架构

```
main.cpp              - 入口、资源加载、游戏主循环
├── scene_manager.h   - 场景状态机管理
├── menu_scene.h      - 主菜单场景
├── selector_scene.h  - 角色选择场景
├── game_scene.h      - 游戏战斗场景
├── player.h          - 玩家基类（含物理、动画、输入）
│   ├── peashooter_player.h  - 豌豆射手子类
│   └── sunflower_player.h   - 向日葵子类
├── bullet.h          - 子弹基类
│   ├── pea_bullet.h         - 豌豆子弹
│   ├── sun_bullet.h         - 日光子弹（带重力）
│   └── sun_bullet_ex.h      - 日光特殊子弹
├── animation.h       - 帧动画系统
├── atlas.h           - 图集管理
├── camera.h          - 摄像机（含震动）
├── vector.h          - 2D 向量数学库
├── timer.h           - 定时器
├── particle.h        - 粒子效果
├── platform.h        - 平台碰撞体
├── status_bar.h      - 血条 UI
└── util.h            - 工具函数（Alpha 混合、图像翻转）
```

## 技术要点

- **面向对象 + 继承多态**：Player/Bullet 基类 + 派生类实现不同角色/子弹行为
- **场景状态机**：SceneManager 管理场景切换（菜单→选择→战斗）
- **物理系统**：重力、跳跃、平台碰撞检测
- **自定义 Vector2 数学库**：运算符重载、归一化、点乘
- **Atlas 图集**：批量加载帧动画资源
- **Camera 震动**：随机偏移模拟打击感
- **定时器系统**：可配置的单次/循环回调计时器
- **像素级图像处理**：水平翻转、描边效果

## 环境要求

- **操作系统**：Windows
- **IDE**：Visual Studio 2019+
- **依赖库**：[EasyX 图形库](https://easyx.cn/)

## 如何运行

1. 安装 [EasyX](https://easyx.cn/) 图形库
2. 用 Visual Studio 打开 `pvp.sln`
3. 编译并运行项目
