# ⚔️ 原神启动 (Genshin Start)

基于 C++ 和 EasyX 图形库实现的原神风格 2D 动作小游戏。

## 功能特性

- 🎮 键盘方向键控制角色（派蒙）移动，支持八方向
- 🐗 敌人（野猪）从四面八方随机刷新，自动追踪玩家
- 🔥 三颗环绕子弹自动旋转攻击，带正弦波动效果
- 💀 碰撞检测：子弹消灭敌人 / 敌人碰到玩家则 Game Over
- 🎵 背景音乐和音效支持
- 🖼️ 精灵动画系统，角色和敌人都有左右朝向动画
- 🌫️ 角色和敌人脚下有阴影效果
- 🎞️ 144 FPS 帧率控制，画面丝滑

## 技术要点

- **Animation 类**：通用帧动画播放系统，支持可配置帧数和帧间隔
- **Alpha 混合**：使用 `AlphaBlend` 实现 PNG 透明通道渲染
- **面向对象设计**：Player、Enemy、Bullet 各为独立类
- **双缓冲绘图**：防止画面闪烁
- **向量归一化移动**：斜向移动速度一致

## 环境要求

- **操作系统**：Windows
- **IDE**：Visual Studio 2019+
- **依赖库**：[EasyX 图形库](https://easyx.cn/)

## 如何运行

1. 安装 [EasyX](https://easyx.cn/) 图形库
2. 用 Visual Studio 打开 `genshin_start.sln`
3. 编译并运行项目
4. 方向键移动角色，ESC 退出游戏

## 操作说明

| 按键 | 功能 |
|------|------|
| ← → ↑ ↓ | 移动角色 |
| ESC | 退出游戏 |
