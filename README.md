## 井字棋 (Tic-Tac-Toe)

使用 C++ 和 EasyX 图形库实现的经典井字棋游戏，支持双人对战。

### 功能
- 600×600 像素的图形界面
- 鼠标点击落子
- 自动判断胜负和平局
- 60 FPS 帧率控制

### 环境要求
- Windows 操作系统
- Visual Studio（推荐 2019 或更新版本）
- [EasyX 图形库](https://easyx.cn/)
