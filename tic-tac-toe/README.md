基于 C++ 和 EasyX 图形库实现的经典井字棋游戏。

## 功能特性

- 🖱️ 鼠标点击落子，操作简单直观
- ⚔️ 双人对战模式（X 先手，O 后手）
- 🏆 自动判断胜负与平局，弹窗提示结果
- 🎞️ 60 FPS 帧率控制，画面流畅

## 技术要点

- 使用 EasyX `graphics.h` 进行图形绘制
- 双缓冲绘图（`BeginBatchDraw` / `FlushBatchDraw`）防止画面闪烁
- 消息循环处理鼠标输入事件

## 环境要求

- **操作系统**：Windows
- **IDE**：Visual Studio 2019+
- **依赖库**：[EasyX 图形库](https://easyx.cn/)

## 如何运行

1. 安装 [EasyX](https://easyx.cn/) 图形库
2. 用 Visual Studio 打开 `demo.sln`
3. 编译并运行项目

## 游戏规则

1. 游戏窗口为 600×600 像素的 3×3 棋盘
2. 玩家 X 先手，点击空格子落子
3. 双方交替落子，先连成三子（横、竖、斜）者获胜
4. 若棋盘填满且无人获胜，则为平局
